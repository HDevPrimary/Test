# Test Repository

Este repositorio contiene un ejemplo simple con un archivo HTML.

## Archivos

- `espacio9.html` – página con un mensaje de **Hola Mundo** y estilo centrado.

## Uso

1. Clona este repositorio en tu máquina:
   ```bash
   git clone https://github.com/HDevPrimary/Test.git
   cd Test
   ```
2. Abre el archivo `espacio9.html` en tu navegador:
   ```bash
   xdg-open espacio9.html  # o simplemente haz doble clic en el archivo
   ```

> También puedes habilitar GitHub Pages en la rama `main` para servir
> automáticamente este archivo desde `https://hdevprimary.github.io/Test/espacio9.html`.
